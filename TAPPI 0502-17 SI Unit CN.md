# Papermaker's Formulas 公式详解（TAPPI TIP 0502-17）

> 本文件基于 TAPPI TIP 0502-17 文档整理，提取所有公式条目，并提供中英文对照与数学表达式。适用于造纸工程师、研究人员及自动化建模参考。

***

## 1. 储罐尺寸和容量 (Tank sizing and capacity)

| 公式                                                                       | 描述                                                                                        |
| :----------------------------------------------------------------------- | :---------------------------------------------------------------------------------------- |
| $t = \text{Volume} \times B.D./100$  $\text{Volume} = t \times 100/B.D.$ | 其中:  $t = \text{公吨}$  $\text{Volume} = \text{储罐体积 (m}^3\text{)}$  $B.D. = \text{纸浆干度百分比}$ |

## 2. 液压泵功率 (Hydraulic pump power)

| 公式                                      | 描述                                                   |
| :-------------------------------------- | :--------------------------------------------------- |
| $\text{Power (kW)} = Q \times H /60000$ | 其中:  $H = \text{泵压差 (kPa)}$  $Q = \text{流量 (l/min)}$ |

In centrifugal pumps or blowers -
A. Capacity varies directly with speed
B. Head varies as the square of speed
C. Power varies as the cube of speed

## 3. 管道和渠道流速 (Pipeline and channel velocity)

| 公式                                                                                                                                       | 描述                                                                                 |
| :--------------------------------------------------------------------------------------------------------------------------------------- | :--------------------------------------------------------------------------------- |
| $V = \text{流速 (m/s)}$  $Q = \text{流量 (L/s)}$  $k\_1 = 0.001$  $k\_2 = 0.001$  $r = \text{管道内半径 (m)}$  $A = \text{管道或渠道截面积 (m}^2\text{)}$ | $V = Q \times k\_1 / (PI\*r^2)$  $V = Q \times k\_2 / A$  筛网至流浆箱适用范围：2.1 至 4.3 m/s |

Screen to headbox acceptable range is 2.1 to 4.3 m/s.
Note: These formulas are for savealls and general pipe flow, since there is no orifice coefficient included

## 4. 堰流量 (Weir water flows)

### 带端部收缩的矩形堰 (Rectangular weir with end contractions)

| 公式                                                                                                                                                               | 描述                                                                                                                                                                         |
| :--------------------------------------------------------------------------------------------------------------------------------------------------------------- | :------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| $Q = 2/3 \times C\_d \times \sqrt{2g} \times L \times H^{3/2}$  $C\_d = 0.622 \times (1 - 0.2 \times H/L)$  $Q = 1.837 \times (L - 0.2 \times H) \times H^{1.5}$ | 其中:  $Q = \text{流量 (m}^3\text{/s)}$  $L = \text{堰口长度 (m)}$ （应为 H 的 4–8 倍）  $H = \text{堰上压头高度 (m)}$ （堰口后约 2 m 处）  $g = 9.81 \text{ m/s}^2$  $a = \text{至少3H（流浆箱侧壁到唇板开口距离）}$ |

### 带端部收缩的三角形堰 (Triangular notch weir with end contractions)

| 公式                                                      | 描述                                                                                                                                                        |
| :------------------------------------------------------ | :-------------------------------------------------------------------------------------------------------------------------------------------------------- |
| $Q = 4/15 \times C \times \sqrt{2gH} \times L \times H$ | 其中:  $Q = \text{流量 (m}^3\text{/s)}$  $L = \text{堰口长度 (m)}$  $H = \text{堰上压头 (m)}$  $g = 9.81 \text{ m/s}^2$  $C= 0.57 \text{ m/s}^2$  $a= \text{不少于3/4L}$ |

针对90度缺口notch
$$Q = 1.3466 \times H^{2.5}$$

针对60度缺口notch
$$Q = 0.7776 \times H^{2.5}$$

## 5. 理论压头（达到目标射流速度所需流浆箱近似压力）(Theoretical head)

| 公式                                       | 描述                             |
| :--------------------------------------- | :----------------------------- |
| $\text{压头 (m H}\_2\text{O)} = V^2/70610$ | 其中:  $V = \text{射流速度 (m/min)}$ |

## 6. 近似射流速度 (Approximate spouting velocity)

| 公式                          | 描述                                                               |
| :-------------------------- | :--------------------------------------------------------------- |
| $V = 265.7 \times \sqrt{h}$ | 其中:  $V = \text{射流速度 (m/min)}$  $h = \text{压头 (m H}\_2\text{O)}$ |

## 7. 流浆箱单位宽度流量（飘片法）(Headbox flow rate per unit width - slice method)

| 公式                              | 描述                                                                                          |
| :------------------------------ | :------------------------------------------------------------------------------------------ |
| $Q = C\_c \times V \times S.O.$ | 其中:  $V = \text{射流速度 (m/min)}$  $S.O. = \text{slice开度 (mm)}$  $C\_c = \text{orifice（收缩）系数}$ |

| 类型       | $C\_c$ |
| :------- | :----- |
| 喷嘴       | 0.95   |
| 当为A 型低角度 | 0.75   |
| 当为B 型高角度 | 0.70   |
| 当C 型垂直   | 0.60   |

## 8. 流浆箱单位宽度流量近似计算（浓度法）(Approximate headbox slice flow rate per unit width - consistency method)

| 公式                                                                                 | 描述                                             |
| :--------------------------------------------------------------------------------- | :--------------------------------------------- |
| $Q = (\text{每天绝干量BDMT/流浆箱宽度m}) \times 70 \* (1.5-\text{网案浓度}) / (1.5\*\text{净浓度})$ | 其中:  $\text{净浓度} = \text{流浆箱浓度} - \text{网案浓度}$ |

## 9. 卫生纸流浆箱单位宽度流量 (Tissue headbox flow rate per unit width)

| 公式                  | 描述                                                                                   |
| :------------------ | :----------------------------------------------------------------------------------- |
| $L = V \times T.O.$ | 其中:  $V = \text{射流速度 (m/min)}$  $T.O. = \text{喉口开度 (mm)}$  注：假设收缩（orifice孔口）系数 = 1.0 |

## 10. 流浆箱自由射流长度 (Headbox free jet length)

| 公式                                                                           | 描述                                                                                                                                                                                                                    |
| :--------------------------------------------------------------------------- | :-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| $x = (v \times \cos A) /35305\* (\sqrt{v^2 \* \sin^2 A + 70610h}-v\*\sin A)$ | 其中:  $v = \text{初始射流速度 (m/min)}$  $A = \text{射流角度 (度)}$  $g = 9.807 \text{ m/s}^2$  $h = \text{唇板尖端至网面高度 (m)}$  $x = \text{射流长度（唇板至着网点）(m)}$  注：a) 适用于水平着网面；b) 射流向下时 A 取正值；c) 射流角度估算参见 TIPs 0410-02, 0410-03, 0410-04 |

## 11. 流量/吨数/浓度关系 (Flow/tons/consistency relationship)

| 公式                                 | 描述                                                                     |
| :--------------------------------- | :--------------------------------------------------------------------- |
| $t/d = C \times Q \times 4.1727/K$ | 其中:  $C = \text{浓度 (%)}$  $Q = \text{流量 (l/min)}$  $K = \text{温度相关系数}$ |

| 温度 (°C) | $K$   |
| :------ | :---- |
| 37.8    | 16.76 |
| 48.9    | 16.83 |
| 60.0    | 16.93 |

\text{留着率 %} = \[(\text{流浆箱浓度} - \text{网案浓度}) / \text{流浆箱浓度}] \times 100

$$

## 13. 成形网上纸料层近似厚度 (Approximate stock thickness on forming fabric)

| 公式                                           | 描述                                                                                                                                                                            |
| :------------------------------------------- | :---------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| $T = (\text{BW} / (R \times C))/10000/(J/W)$ | 其中:  $T = \text{网案上纸料层厚度 (cm)}$  $\text{BW} = \text{定量 (g/m}^2\text{)}$  $C = \text{浓度 (%)}$  $R = \text{该点至机尾留着率 (%)}$  $J/W = \text{射流/网速比} = 1.0$（缝口处除外）  注：流浆箱缝口处结果为缩颈后厚度 |

## 15. 成形 – 刮刀脉冲频率 (Formation – blade pulse frequency)

$$f = V/(k\*\lambda)$$

| 参数                                                                   | 说明                                  |
| :------------------------------------------------------------------- | :---------------------------------- |
| $V = \text{网速 (m/min)}$  $\lambda = \text{刮刀间距（尖端至尖端）(m)}$  $k = 60$ | 优化频率因纸种而异：通常$f > 60$ Hz，最高可达 150 Hz |

## 16. 长网摇振数 (Fourdrinier shake number)

$$\text{摇振数} = (\text{振幅} \times \text{频率}^2) / \text{网速}$$

| 参数                                  | 说明                                                      |
| :---------------------------------- | :------------------------------------------------------ |
| 振幅（冲程长度）(mm)  频率（冲程/分钟）  网速 (m/min) | 成形改善通常在摇振数 >2500 时出现；建议目标值：4200–5000； >5000 可能有益，但设备常受限 |

## 17. 案辊转速 (Dandy roll rotational speed)

$$\text{RPM} = \text{网速} / \text{辊子直径} /3.142$$

| 参数                                 | 说明                                  |
| :--------------------------------- | :---------------------------------- |
| RPM = 转速 (转/分)  网速 (m/min)  直径 (m) | 目标值：125–150 转/分；公式为近似值（考虑案辊与网面可能打滑） |

## 18. 气体定律（常用于真空系统）(Gas laws)

$P \times V = \eta \times R \times T$（理想气体定律）
$P\_1 \times V\_1 = P\_2 \times V\_2$（玻意耳定律）
$(P\_1 \times V\_1)/T\_1 = (P\_2 \times V\_2)/T\_2$（联合气体定律）
$V2=(1.0132-p1)/(1.0132-p2)\*V1$, p单位bar，V单位m3/h.

| 参数                                                                                                                                     | 说明     |
| :------------------------------------------------------------------------------------------------------------------------------------- | :----- |
| $P = \text{绝对压力 (bar)} = (\text{表压} + 1.0132)$  $V = \text{气体总体积 m}^3$  $\eta = \text{气体质量 (kg)}$  $T = \text{绝对温度 (K)} = °C + 273.15$ | <br /> |
| R = 气体系数 bar·m^3/(kg·mol·K)                                                                                                            | <br /> |
| Ra = 0.00287                                                                                                                           | <br /> |
| Rw = 0.004614                                                                                                                          | <br /> |

| :----------------------------- | :---------------------------------------------------------------------------------------------- |

## 20. 拖曳负荷 – 常规 (Drag load – conventional)

| 公式                                | 描述                                                                                                                                 |
| :-------------------------------- | :--------------------------------------------------------------------------------------------------------------------------------- |

## 21. 部件拖曳负荷（湿部）(Component drag load - wet end)

| 公式                                | 描述                                                                                                                                                                                                                                        |

| $DL = (V\_n/V\_s - 1)(EM - T\_s)$ | 其中:  $DL = \text{拖曳负荷 (kN/m)}$  $V\_n = \text{织物在 n 点速度 (m/min)}$  $V\_s = \text{织物松边速度 (m/min)}$  $EM = \text{织物弹性模量 (kN/m)} \approx EMr - KT$  $EMr = \text{参考温度 r 下弹性模量}$  $K = \text{模量/温度系数 (kN/m/°C)}$  $T\_s = \text{松边张力 (kN/m)}$ |

## 23. 根据压区印痕宽度确定两辊总中高度变化的近似方法 (Roll crown change from nip impression)

$$C = \[(D\_1 + D\_2) \times (N\_e^2 - N\_c^2)] / \[2 \times (D\_1 \* D\_2)]$$

| 参数                                                                                                                 | 说明               |
| :----------------------------------------------------------------------------------------------------------------- | :--------------- |
| $C = \text{两辊总中高度变化量}$  $N\_e = \text{压区端部宽度}$  $N\_c = \text{压区中部宽度}$  $D\_1 = \text{上辊直径}$  $D\_2 = \text{下辊直径}$ | 单位：统一使用 SI 单位（米） |

## 24. 辊面挠度（通常用于中高度计算）(Roll deflection over face)

## 27. 未涂布纸干燥速率 (Drying rate for uncoated paper)

M=L/E-1
$$RW = 60\*(SBM)/(NA\pi D)$$

其中：
$RW = \text{干燥速率（蒸发水量）}$，$S = \text{车速}$，$B = \text{出烘干部纸页定量（湿基）}$，
$M = \text{每单位干纸蒸发水量（湿基）}$，$N = \text{接触纸页的蒸汽加热烘缸数量}$，
$A = \text{标准面积}$，$D = \text{烘缸直径}$，
$L = \text{离开末缸纸页干度（湿基，较大值）}$，$E = \text{进入首缸纸页干度（湿基，较小值）}$

## 28. 涂布纸干燥速率 (Drying rate for coated paper)

使用公式 #27，但定量和入口干度按以下公式确定：
$B = ((B\_c \times ( P/100)) +W) / ( L /100)$

$$
E = 100 - 100 \left\[ \frac{B\_c \left(1 - \dfrac{P}{100} \right) + W \left( \dfrac{100}{C} - 1 \right)}{B\_c + \dfrac{100W}{C}} \right]
$$

其中：
$B\_c = \text{进入涂布机纸页定量（湿基）}$，$W = \text{干涂布量}$，
$P = \text{进入涂布机纸页干度百分比}$，$C = \text{涂布液固含量（湿基）}$

| 参数                                                                      | 单位     |

## 29. 烘缸 rimming 速度（5英尺和6英尺烘缸）(Rimming speed)

$$\text{Rimming speed (ft/min)} = \[5720 - (2160/D)] \times L^{1/3}$$

其中 $D = \text{辊内径 (ft)}$，$L = \text{冷凝液膜厚度 (ft)}$
注：此为经验公式。使用英制单位计算后转换为 SI（结果 × 0.3048 得 m/min）

## 31. 卫生纸起皱率计算 (Tissue crepe)

常用两种方法（结果不同，不可直接比较）：

## 32. 瞬时产量（卷取）(Instantaneous production rate)

| 公式                                     | 描述                                                                                                                                    |

## 38. 压光辊临界转速 (Critical speed of calender roll)

## 41. Torque

### 扭矩

$$
T = F \times r
$$

其中：

- $ T $：扭矩（Nm）
- $ F $：力（N）
- $ r $：力臂（m）

## 42. Power

### 功率

$$
P = \frac{T \times N}{5252}
$$

其中：

- $ P $：功率（W）
- $ T $：扭矩（Nm）
- $ N $：转速（rpm）
- 5252：单位换算常数

> ✅ 电机功率计算核心公式
